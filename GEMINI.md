# Factorio Idle Machine Mod

## Project Context
This is a Factorio mod for version 2.0. It features an "Idle Machine" that levels up and generates items over time.

## Modding Standards
See [FACTORIO_MODDING.md](./FACTORIO_MODDING.md) for detailed Factorio 2.0 modding standards, patterns, and best practices used in this project.
